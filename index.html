Microsoft Excel

Analyse de données et automatisation de calculs

Environnement

  • Classeur : Le fichier global (.xlsx).
  • Ruban : Outils Accueil, Insertion, Formules...
  • Cellule : Unité de base (ex: B4).

Mise en Forme

  • Format de Cellule : Monétaire, Date, Pourcentage.
  • Recopie : Étendre les séries avec le carré vert.
  • Conditionnelle : Couleurs auto selon les valeurs.

Opérateurs

+ Addition - Soustraction * Multiplication / Division

Statistiques

=SOMME(plage)

=MOYENNE(plage)

=NB.SI(plage ; critère)

Logique

=SI(Test ; Vrai ; Faux)

Ex: =SI(A1>=10; "Admis"; "Recalé")

Microsoft PowerPoint

Le Masque de Diapos

L'outil indispensable : modifiez une fois (logo, police), appliquez partout.
Accès : Affichage > Masque de diapositives.

Formats de diapo

Standard (4:3) ou Grand écran (16:9).

Règle d'or visuelle

"1 Idée = 1 Diapositive"

Transitions

Mouvement entre deux pages (ex: Fondu, Morphose).

Animations

Mouvement d'un objet (Texte, Image).
